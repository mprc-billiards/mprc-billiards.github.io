# 补充说明

### 满贯

MPRC台球俱乐部的满贯类型如下：

- 小满贯：获得全部4种例行赛的冠军
- 大满贯：获得全部4种例行赛的冠军+公开赛冠军
- 金满贯：获得全部4种例行赛的冠军+公开赛冠军+锦标赛冠军
- 全满贯：获得公开赛全部4个分站的冠军（自公开赛进入巡回赛时代）

获得满贯的周期如下：

- 生涯满贯：在整个生涯范围内获得的满贯
- 赛季满贯：在单个赛季范围内获得的满贯
