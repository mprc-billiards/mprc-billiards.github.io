四球友谊赛采用九球的变种规则，将九球缩减为四球。该比赛为友谊赛，不计入积分，部分比赛记录可能丢失。四球比赛可按如下规则计分：

1. 【大  金】开球完成清台（按顺序击打，9号球单独打进），计3分
2. 【小  金】台面上四颗球都在，完成清台（按顺序击打，9号球单独打进），计2分
3. 【黄金九】击打其他号球将9号球打进，计1分
4. 【普  胜】普通地将9号球打进，计1分
