四球追分友谊赛采用九球追分的变种规则，将九球缩减为四球。该比赛为友谊赛，不计入积分，部分比赛记录可能丢失。四球追分友谊赛可按如下规则计分：

1. 【大  金】开球并且按顺序完成清台，计5分
2. 【全球金】台面上全部球都在，按顺序完成清台，计4分
3. 【一九金】台面上1号球和9号球在，按顺序完成清台，计3分
4. 【黄金九】击打其他号球将9号球打进，计2分
5. 【普  胜】普通地将9号球打进，计1分
