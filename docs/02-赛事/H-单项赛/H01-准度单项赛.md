准度单项赛是MPRC台球俱乐部成员之间的准度比拼游戏，包含以下比赛项目：

1. 直线球
2. 角度球
3. 贴库球
4. 翻袋球

计分采用连进分值累加制，各项分数相加得到总分，总分最高者获得“准神”称号。排名优先级如下：

1. 总分更高者，排名靠前；
2. 最大连进数更高者，排名靠前；
3. 总进球数更高者，排名靠前；
4. 加赛一颗球，打进者排名靠前。
