# 普尔台球挑战赛/Pool Challenge

| 届次 | 日期                   | 场地         | 胜者   | 比分 | 负者  |
| ---- | --------------------- | ------------ | ----- | ---- | ----- |
| 1    | 2024.12.05-2024.12.12 | 邱德拔/谈小娱 | 王翰墨 | 4-3 | 姜星宇 |

普尔台球挑战赛是各种普尔台球的混合系列赛，2名选手进行7局4胜制，7场比赛的规则分别如下：

1. 单袋台球（One-Pocket Pool）
2. 翻袋台球（Bank Pool）
3. 残酷台球（Cutthroat Pool）
4. 直球（Straight Pool）
5. 旋转台球（Rotation Pool）
6. 八球（8-Ball Pool）
7. 九球（9-Ball Pool）
