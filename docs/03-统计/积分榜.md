**NOTE:**修正积分用于修正因选手缺席部分例行赛所导致的积分损失，进而更客观地反映选手水平，并非用于排名。计算公式为：
```c
修正积分 = (参加比赛积分上限 == 0) ? 0 : (总积分 * 全部比赛积分上限 / 参加比赛积分上限);
```
