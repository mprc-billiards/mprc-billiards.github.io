# 中式八球公开赛/Chinese 8-Ball Open

| 届次 | 日期                  | 场地         | 冠军   | 亚军   | 季军   | 决赛比分 |
| ---- | --------------------- | ------------ | ------ | ------ | ------ | -------- |
| 1    | 2024.10.18-2024.10.20 | 邱德拔体育馆 | 王翰墨 | 姜星宇 | 魏天昊 | 3-1      |

中式八球公开赛是MPRC台球俱乐部历史最悠久的八球赛事。比赛采用淘汰赛制，抽签决定对阵，直至决出冠军。半决赛采用三局两胜制，决赛采用五局三胜制，季军赛采用单局制。此外，比赛要求至少4名选手参赛。
